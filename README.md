# Bangalore Home Price Prediction 🏠

This project focuses on predicting real estate prices in Bangalore, India, using Machine Learning techniques. It covers the entire data science pipeline, from data cleaning to building a functional predictive model with a user interface.

## 🛠️ Skills & Technologies Applied

* **Data Cleaning:** Handled missing values (NA), removed outliers, and performed data consistency checks.
* **Feature Engineering:** Dimension reduction (Dimensionality Reduction) and data transformation to improve model accuracy.
* **Model Building:** Implemented a **Linear Regression** model using Scikit-Learn.
* **Tools Used:** Python, Pandas, Numpy, Matplotlib, and Scikit-Learn.
* **GUI:** Developed a simple Graphical User Interface to predict prices based on area (Square Feet) and number of bedrooms (BHK).

*Developed as part of my technical studies in Computer Science.*
